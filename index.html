Abstract

This work tackles occlusion issues in Virtual Try-On (VTON).
We taxonomize failures into:

1. Inherent Occlusions ‚Äì ‚Äúghost‚Äù garments from the reference image that remain in the result.

2. Acquired Occlusions ‚Äì distorted human anatomy that visually blocks the new outfit.

To remove both, we propose a mask-free VTON framework with two plug-and-play operations:

- Background Pre-Replacement ‚Äì swaps the background before generation so the model never confuses clothes with body/background, suppressing inherent occlusions.

- Covering-and-Eliminating ‚Äì enforces human-aware semantics, yielding anatomically plausible shapes and thus fewer acquired occlusions.

The operations are architecture-agnostic: drop them into GANs or diffusion models without re-design.
